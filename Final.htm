After looking for awhile i found a few that were using arduino as their main electronic system which suited my design as i woul be using arduino as well in my design. Out of all the desgins i decided to go with one made by daichan blog

The picture above is daichan's design. Using this product as a template i design my own dispenser which fulfilled the requirements of the project
