Embedded Programming

Pedestrian Crossing

The objective

Build and program a pedestrian crossing system

  • stop/go (2leds)
  • pushbutton switch

At start, RED on (no crosssing)

  • press button
  • delay for 5s
  • turn off RED, turn on GREEN
  • delay for 6s
  • turn on RED, turn off GREEN

Possible improvements

  • blinking GREEN man
  • integrate with traffic light for cars

The design

Before making the actual system, i used thinkercad to design my circuit and also simulate my programs which lightens my physical workload

The program

After i am done with the design of the cicuit i can then continue using thinkercad to code and simulate my programs

The device

After checking that the simulation is working as intended, i move on to the actual arduino board and start making the cicuit as designed in thinkercad

Final system

Once the device is done i went ahead and copied my thinkercad code into the arduino software and uploaded the code into my arduino board
